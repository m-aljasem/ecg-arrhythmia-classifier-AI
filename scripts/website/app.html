Streamlit App Coming Soon!

This page will host the live ECG classification application once deployed to Streamlit Cloud.

📋 Deployment Instructions

To embed your Streamlit app on this page, follow these steps:

  1. Deploy to Streamlit Cloud:
    • Go to share.streamlit.io
    • Connect your GitHub repository
    • Select the app.py file from the project
    • Click "Deploy"
  2. Get Your App URL:
    • After deployment, copy your app URL
    • It will look like: https://your-app.streamlit.app
  3. Embed in This Page:
    • Open website/app.html
    • Find the <div class="app-placeholder"> section
    • Replace it with: <iframe class="app-frame" src="YOUR_STREAMLIT_URL?embedded=true"></iframe>
💡 Pro Tip: Add ?embedded=true to your Streamlit URL to hide the default toolbar and make it blend better with your website!
⚠️ Note: Make sure your Streamlit app settings allow iframe embedding. Go to Settings → General → Enable "Allow embedding" in Streamlit Cloud.

💻 Example Embed Code

Replace the placeholder div with this iframe code:

<!-- App Container -->
<div class="app-container">
    <iframe 
        class="app-frame" 
        src="https://your-app.streamlit.app?embedded=true"
        title="ECG Classification Application"
        allow="camera; microphone"
    ></iframe>
</div>
